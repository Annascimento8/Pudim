# 🍮 A Lenda do Pudim - Landing Page

Esta é uma página web moderna e responsiva dedicada à história, curiosidades e à receita clássica do pudim de leite condensado. O projeto foi desenvolvido como uma atividade prática de Web Design, focando em semântica HTML5 e estilização personalizada com CSS3.

## 🚀 Demonstração

Você pode visualizar o projeto online através do link abaixo:
**https://atividad-pudim.netlify.app/**

---

## 🎨 Sobre o Projeto

O objetivo foi transformar um código simples em uma experiência visual gastronômica, utilizando elementos que remetem ao conforto e à doçura da culinária artesanal.

### 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização interna, layouts com `max-width` e centralização inteligente.
* **Google Fonts**: 
    * *Dancing Script*: Para títulos com aspecto de escrita à mão.
    * *Quicksand*: Para uma leitura leve e moderna nos parágrafos.
* **Netlify**: Hospedagem e deploy contínuo.

---

## 🍱 Funcionalidades

- [x] **Design Responsivo**: Adaptado para visualização em computadores e dispositivos móveis.
- [x] **Seção de Curiosidades**: Informações históricas sobre a origem do doce.
- [x] **Receita Completa**: Ingredientes e modo de preparo organizados em listas.
- [x] **Social**: Botões de interação com redes sociais.
- [x] **UX/UI**: Uso de sombras suaves, bordas arredondadas e contraste de cores creme e marrom (caramelo).
